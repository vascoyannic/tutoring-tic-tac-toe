Konsolen-befehle zum merken:
npm i -g <paket> // Installiert ein Paket global
npm i --save <paket> // Installiert ein Paket als Dependency
npm i --save-dev <paket> // Installiert ein Paket als Dev-Dependency
npm run <command> // Führt den Befehl mit Alias command aus
    - npm run start // Startet den dev-server
    - npm run build // Baut HTML, JS, CSS
npm i // Installiert aktuelle Pakete nach der package.json
